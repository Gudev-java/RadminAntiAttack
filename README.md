# RadminAntiAttack Filter (RAA Filter)

Короче, этот мод делает так, чтоб всякие боты-сканеры и прочие гриферские софтины не видели, кто сидит на сервере.

### Чо он делает:
Когда кто-то пингает сервак (смотрит список серверов), мод перехватывает пакет и затирает там список игроков (player sample).
- **Счетчик игроков** (типа 5/20) остается рабочим, чтоб игроки не пугались.
- **Ники игроков** скрываются. Боты не смогут спарсить твой ник и начать атаку по подмене сессии.

### Установка:
1. Ставишь Fabric.
2. Кидаешь этот мод в папку `mods`.
3. Всё, боты сосут бибу.

### Совместимость:
- Майн 1.21 и выше (проверено на 1.21.11).
- Лоадер от 0.15.0 и выше.
- Работает на выделенных серверах и на "Открыть для сети".
